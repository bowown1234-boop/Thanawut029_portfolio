สร้างหน้าโปรไฟล์ผู้ใช้แบบ Responsive ด้วย HTML

คุณได้รับมอบหมายให้สร้างหน้าโปรไฟล์ผู้ใช้แบบ Responsive โดยใช้เฉพาะ HTML ในการจัดโครงสร้างเนื้อหา เป้าหมายคือการใช้แท็ก HTML5 เชิงความหมาย (semantic HTML5) เพื่อให้โครงสร้างของหน้าเว็บมีความหมาย และเป็นระเบียบ นอกจากนี้คุณต้องออกแบบให้หน้าเว็บ พร้อมสำหรับการปรับขนาดหน้าจอ (responsive) โดยไม่จำเป็นต้องใช้ CSS ในขั้นตอนนี้ สิ่งที่โจทย์ต้องการ: ส่วนหัวของหน้า: ใช้แท็ก <header> สำหรับส่วนหัวของหน้า ซึ่งควรมีชื่อโปรไฟล์ และรูปภาพโปรไฟล์ เนื้อหาหลัก: ใช้แท็ก <main> สำหรับเนื้อหาหลักของโปรไฟล์ ภายใน <main>: ให้ใช้แท็ก <section> อย่างน้อย 2 ส่วน ส่วนหนึ่งสำหรับข้อมูลส่วนตัว เช่น ชื่อ อาชีพ และเมือง และอีกส่วนหนึ่งสำหรับ ประวัติหรือความสนใจ บทความหรือโพสต์: ใช้แท็ก <article> ภายใน <section> สำหรับข้อมูลที่เกี่ยวข้องกับ โพสต์หรือบทความของผู้ใช้ (สมมติว่ามี) ข้อมูลเสริม: ใช้แท็ก <aside> สำหรับข้อมูลเสริม เช่น ลิงก์โซเชียลมีเดีย หรือรายชื่อเพื่อน ส่วนท้ายของหน้า: ใช้แท็ก <footer> สำหรับส่วนท้ายของหน้า ซึ่งอาจมีข้อมูลลิขสิทธิ์ หรือลิงก์ติดต่อ รูปภาพ: ใช้แท็ก <img> สำหรับรูปภาพโปรไฟล์ และต้องใส่ alt attribute ที่เหมาะสม เมนูนำทาง: ใช้แท็ก <nav> สำหรับเมนูนำทางหลัก ของโปรไฟล์ (ถ้ามี) รายการข้อมูล: ใช้แท็กรายการ เช่น <ul> และ <li> สำหรับแสดงข้อมูลต่าง ๆ เช่น รายชื่อความสนใจ หรือลิงก์โซเชียลมีเดีย การรองรับ Responsive: ตรวจสอบให้แน่ใจว่า แท็ก <meta name="viewport"> ถูกตั้งค่าอย่างถูกต้อง ภายใน <head> เพื่อรองรับการแสดงผล บนอุปกรณ์หลายขนาดหน้าจอ

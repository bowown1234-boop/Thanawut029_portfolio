สร้างหน้าแสดงรายการสินค้าด้วย HTML เชิงความหมาย

คุณได้รับมอบหมายให้สร้างหน้าเว็บ สำหรับแสดงรายการสินค้าต่าง ๆ ของร้านค้า โดยเน้นการใช้แท็ก HTML ที่มีความหมาย (Semantic HTML) เพื่อให้โครงสร้างของหน้าเว็บมีความชัดเจน เข้าถึงง่าย และดีต่อ SEO โดยไม่ต้องใช้ CSS ในขั้นตอนนี้ หน้าเว็บควรแสดงข้อมูลสินค้า อย่างน้อย 3 ชิ้น สิ่งที่โจทย์ต้องการ: ส่วนหัวของหน้าเว็บ (Header): สร้างส่วนหัวของหน้าเว็บ โดยใช้แท็ก <header> ภายในต้องมีชื่อร้านค้า เช่น “ร้านค้าของฉัน” และเมนูนำทาง (Navigation) อย่างน้อย 2 รายการ เช่น “หน้าแรก” และ “สินค้า” เมนูนำทางต้องใช้แท็ก <nav> และ <ul> ส่วนหลักของหน้าเว็บ (Main Content): สร้างส่วนหลักของหน้าเว็บ โดยใช้แท็ก <main> ภายในส่วนหลัก: ให้สร้างส่วนสำหรับแสดงรายการสินค้า โดยใช้แท็ก <section> รายการสินค้า: ภายใน <section> ให้สร้างรายการสินค้า อย่างน้อย 3 ชิ้น โดยสินค้าแต่ละชิ้น ควรแยกออกจากกันอย่างชัดเจน สามารถใช้แท็ก <article> หรือ <div> ที่เหมาะสม สำหรับสินค้าแต่ละชิ้น องค์ประกอบของสินค้าแต่ละชิ้น: สินค้าแต่ละชิ้น ต้องประกอบด้วยข้อมูลดังนี้ - รูปภาพสินค้า ใช้แท็ก <img> และต้องมี alt attribute ที่เหมาะสม - ชื่อสินค้า ใช้แท็กส่วนหัว เช่น <h2> หรือ <h3> - คำอธิบายสั้น ๆ ของสินค้า ใช้แท็ก <p> - ราคาสินค้า - ปุ่ม “เพิ่มลงตะกร้า” ใช้แท็ก <button> ส่วนท้ายของหน้าเว็บ (Footer): สร้างส่วนท้ายของหน้าเว็บ โดยใช้แท็ก <footer> ภายในอาจมีข้อมูลลิขสิทธิ์ หรือข้อมูลติดต่อ แสดงอยู่ด้านล่างสุดของหน้า
